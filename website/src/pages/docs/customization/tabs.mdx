This section provides a quick start guide for the tabs component.

### Tabs

The Tabs component is useful for displaying content in a tabbed format. This can be particularly useful for showing different code snippets or views.

<Tabs
  tabs={[
    { title: "Tab 1", content: "Content for Tab 1" },
    { title: "Tab 2", content: "Content for Tab 2" },
    { title: "Tab 3", content: "Content for Tab 3" },
  ]}
/>

export default function MDXPage({ children }) {
  return <DocsLayout>{children}</DocsLayout>;
}
